Word Prediction Application

Yash Deshpande
30th July, 2018

This presentation acts as a pitch for an application that implements the auto-complete functionality, by taking a set of words as input from the user, and suggesting what the next word should be. This application has been created as a culmination to the Data Science Specialization, under Coursera and Johns Hopkins University, in partnership with SwiftKey.

The Objective

The goal of this capstone project was to build a Shiny application that predicts the next word on the basis of a set of input words.

This project leverages the various methods learnt during the course of the Coursera Data Science specialization, and is a step-by-step build of a data product. Exploratory analyses, statistical modelling, and learning techniques have been implemented using various R packages.

Data from the corpus provided HC Corpora has been used to create a frequency dictionary of n-grams (a set of n words), and this dictionary has been used to predict the next word.
